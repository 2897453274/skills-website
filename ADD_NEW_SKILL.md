# 添加新技能指南

## 📋 如何添加新学习的技能

### 方法一：手动编辑JSON文件
1. 打开 `skills-inventory.json`
2. 在 `skillDetails` 对象中添加新技能
3. 更新 `totalSkills` 计数
4. 更新 `lastUpdated` 时间戳

### 方法二：使用技能管理器（推荐）
1. 技能管理器会自动处理数据更新
2. 提供统一的API接口
3. 支持搜索、分类、统计功能

### 方法三：通过Web界面
未来可以开发一个管理界面来可视化添加技能

## 🎯 技能数据格式

```json
{
  "skill-name": {
    "level": "中级",
    "experience": "1年",
    "description": "技能描述信息"
  }
}
```

## 🔄 自动更新机制

网站包含自动更新功能：
- **每日检查**: 每天自动检查技能更新
- **手动触发**: 可以手动触发立即更新
- **版本控制**: 保持数据的一致性和完整性

## 📊 统计信息

添加新技能后，统计信息会自动更新：
- 总技能数量
- 按分类分布
- 按等级分布
- 最后更新时间

## 🚀 最佳实践

1. **及时添加**: 学习新技能后立即记录
2. **详细描述**: 提供清晰的技能描述
3. **正确分类**: 选择适当的技能分类
4. **客观评级**: 真实评估技能水平
5. **定期回顾**: 定期检查并更新技能信息
